Check out this HTML5 Canvas below!

Hover over a mesh to change its color.

You can create this same Three.js scene in 4 steps:

  1. Configure Three.js: Create and configure the Scene object with a Camera and a Renderer.
  2. Mesh Creation: Create a new Mesh using a GeometryObject and a Material, then Add it to your scene.
  3. Make a Render Loop: Define a JavaScript function that tells our app what code to run on every single frame (usually 60 per second).
  4. Call the Render Loop: Call your render() function to start the animation!
